## тест-кейс: Переход на страницу решений компании и сортировка по цене

**расположение:**  
project\tests\test_cases\vivat_solutions_sorting\test_vivat_solutions_sorting.py

### Предусловия
- Главная страница открыта

### Шаги

| шаг | ожидаемый результат |
|---|---|
| Нажать на кнопку «Решения VIVAT-IQ» в шапке страницы | Открывается страница "Решения VIVAT-IQ" |
| Проверить отображение товаров | На странице отображаются товары компании |
| Выбрать сортировку по цене (сначала дешевые) | Решения отсортированы по возрастанию цены |
| Выбрать сортировку по цене (сначала дорогие) | Решения отсортированы по убыванию цены |
