## тест-кейс: Проверка подсчета итоговой суммы товаров в корзине

**расположение:**  
project\tests\test_cases\cart_total_price\test_cart_total_price.py

### Предусловия
- Главная страница открыта  

### Шаги

| шаг | ожидаемый результат |
|---|---|
| Проверить состояние иконки корзины | Рядом с иконкой корзины отображается число 0 |
| Нажать на иконку корзины | Открывается страница корзины |
| Проверить состояние корзины | Отображается сообщение «Ваша корзина пуста» |
| Нажать кнопку "В каталог" | Происходит переход на страницу "Решения 1С" |
| Нажать на карточку "Торговые решения", нажать на карточку товара "1С:Розница 8 ПРОФ", выбрать количество товаров 5, нажать на кнопку "в корзину" | Рядом с иконкой корзины отображается число 1 |
| Нажать на иконку корзины | Открывается страница корзины |
| Проверить товар в корзине | Отображаются правильные: наименование, цена, количество и сумма |
| Проверить итоговую сумму | Отображается корректная итоговая сумма товаров |
| Нажать кнопку "Очистить корзину" | Отображается сообщение «Ваша корзина пуста» |
