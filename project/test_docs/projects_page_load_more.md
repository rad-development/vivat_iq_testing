## тест-кейс: Проверка работы кнопки «Загрузить еще» на странице проектов

**расположение:**  
project\tests\test_cases\projects_page_load_more\test_projects_page_load_more.py

### Предусловия
- Главная страница открыта

### Шаги

| шаг | ожидаемый результат |
|---|---|
| Нажать на кнопку «Проекты» в шапке страницы | Открывается страница «Проекты» |
| Проверить, что отображены проекты | На странице отображается список проектов |
| Нажать кнопку «Загрузить еще» | Количество отображаемых проектов увеличивается |
