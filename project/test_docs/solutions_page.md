## тест-кейс: Переход на страницу «Решения 1С»

**расположение:**  
project\tests\test_cases\solutions_page\test_solutions_page.py

### Предусловия
- Главная страница открыта

### Шаги

| Шаг | Ожидаемый результат |
|---|---|
Нажать на кнопку «Решения 1С» в шапке сайта | Открывается страница «Решения 1С» и отображается список карточек решений |
Проверить список решений на странице | Список решений не пустой, заголовки карточек читаемы |
Кликнуть по первой карточке решения | Открывается страница группы продуктов с правильным названием группы |
Проверить список продуктов в группе | Список продуктов не пустой, заголовки читаемы |
Кликнуть по первой карточке продукта | Открывается страница продукта |
Проверить название продукта | Название продукта совпадает с выбранным в группе продуктов |
