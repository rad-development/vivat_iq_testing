
## тест-кейс: Переход на страницу услуг и проверка списка услуг

**расположение:**  
project\tests\test_cases\services_page_navigation\test_services_page_navigation.py

### Предусловия
- Главная страница открыта

### Шаги

| шаг | ожидаемый результат |
|---|---|
| Нажать на кнопку «Услуги» в шапке страницы | Открывается страница «Услуги» |
| Проверить отображение списка услуг | На странице отображается список услуг |
| Нажать на ссылку (стрелку) первой услуги | Открывается страница выбранной услуги |
| Сравнить название услуги из списка услуг с названием на странице услуги | Названия совпадают |
