# VIVAT-IQ
### Описание

Этот репозиторий содержит автоматизированные UI-тесты для веб-приложения vivat-iq:
`https://www.vivat-iq.ru/`

Автотесты реализованы на языке **Python** с использованием фреймворка **pytest**, библиотеки **Selenium WebDriver**, вспомогательной библиотеки **py-selenium-auto** и паттерна проектирования **Page Object**.

## Установка и запуск
  
### Установка зависимостей

`pip install -r requirements.txt`

### Запуск тестов

`pytest -v --html=Log/report.html --self-contained-html`

В результате в папке `/Log` будет создан HTML-отчёт о результатах тестирования report.html, который можно открыть в браузере.

### Конфигурация

Настройки браузера находятся в файле:
`\tests\resources\settings.json`

## Реализованные автотесты

- cart_total_price — проверка итоговой стоимости в корзине
- home_page_coop_form — проверка формы "начать сотрудничество" на главной странице
- no_html_leaks_on_pages — проверка отсутствия "утечек" HTML на страницах
- projects_page_load_more — проверка кнопки "Показать ещё"
- services_page_navigation — проверка навигации на странице услуг
- solutions_page — проверка страницы решений 1С
- vivat_solutions_sorting — проверка сортировки на странице "решения VIVAT-IQ"

### Тест-кейсы находятся в папке `/test_docs`

## Технологии
- Python (3.12.0): основной язык разработки
- py-selenium-auto: вспомогательная библиотека для упрощения работы с Selenium
- Selenium WebDriver: инструмент для автоматизации браузера, управляет браузером и имитирует действия пользователя (клики, ввод текста и т.д.)
- Pytest: фреймворк для написания и запуска тестов
- Pydantic: используется для работы с конфигурациями, тестовыми данными и валидации данных.

Полный список использованных библиотек в файле `\requirements.txt`
